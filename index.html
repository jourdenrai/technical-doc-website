Setting up a Node Development Environment



Introduction

  • Node is an open-source, cross-platform, server environment.

  • It allows you to run JavaScript code outside of a web browser

  • It has some great features that make it an attractive choice for building server-side application middle tiers, including web servers and web services for platform APIs.


    • Installing Node.js

    Before we get going, let's make sure you have node installed.

    Windows or macOS

    Go to the node website here and download the "Recommended for most users" installer.

    Ubuntu 18.04

    The easiest way to install the most recent LTS version of Node 10.x is to use the package manager to get it from the Ubuntu binary distributions repository.

    Run these two commands in your terminal.

      curl -sL https://deb.nodesource.com/setup_10.x | sudo -E bash -
      sudo apt-get install -y nodejs

    npm

  • npm is a package manager for the JavaScript programming language.

  • You can manually use npm to separately fetch each needed package.

  • This isthe most important tool for working with Node.js applications.

    • Use the npm init command to create a package.json file for your application.

      npm init

    package.json is used to give information to npm that allows it to identify the project as well as handle the project's dependencies.

    Your terminal will prompt you to enter values for the package.json file.
    Here's an example:

      {
       "name": "myapp",
       "version": "1.0.0",
       "description": "",
       "main": "server.js",
       "scripts":  {
       "test": "echo \"Error: no test specified\" && exit 1"
       },
       "author": "",
       "license": "ISC"
       }

    Great! You can now use npm to install whatever packages you want!


    Express.js

  • Express is a minimal and flexible Node.js web application framework that provides a robust set of features for web and mobile applications.

    • The hard part is over, to install Express with npm enter this command

      npm install express --save

    See that wasn't soo bad!

    Node & express are so easy enough that you can get a basic web server to serve "Hello World!" in about 12 lines of code

    Create a new file called server.js

    Add this inside your new file:

     const express = require('express');

     const app = express();
     const port = process.env.PORT || 3000;

     app.listen(port, () => {
     console.log(`listening on port ${ port }`);
     });

    Save your file and run your express app by typing this command:

     node server.js

    You should see this in the console:

     listening on port 3000

    Tada! Your server is now up and running!


    Hello World!

    Let's serve a simple "Hello World!" to your web server page.

    Create an app.get function above your app.listen function.

     app.get('/', (req, res) => {
     res.send("Hello, world!");
     });

    Save your file and open up a browser and type localhost:3000

    You should now see "Hello World!" on your web server page


    Conclusion

    You now have a Node development environment up and running on your computer that can be used for creating Express web applications. You've also seen how npm can be used to import Express into an application.

    We’ve only just scratched the surface of Node and Express. There’s a lot more to learn about, including session management, token authentication, API design, etc…
